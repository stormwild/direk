# direk

Subscription Video On Demand

## Pre-requisites

- PHP 7
- Sqlite
- FFMpeg

## Setup

Git clone the repository

```
git@github.com:stormwild/direk.git
```

Run Composer

```
composer install
```

Run npm

```
npm run dev
```

## Run

To run:

```
php artisan serve
```
